# react-electron-boilerplate
A boilerplate for creating project that based on Electron and React.js  

### Prerequisites

### Install

```bash
  npm install
  yarn install
```  
### Run

```bash
  npm run dev
  yarn dev
```  
### Testing

```bash
  npm run test
  yarn test
```
